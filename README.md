LogiQ — умный AI-ассистент для микробизнеса
•  Ссылка на работающего бота:
@LogiQ_bot
•  Цель:

Помогать владельцам и будущим владельцам небольших бизнесов быстрее и эффективнее решать повседневные задачи.

•  Функции и возможности:

• Помощь в определении направления — с чего начать бизнес
• Проведение первичной диагностики текущего состояния дела
• Советы по маркетингу, финансам и процессам

•  Как пользоваться:

1. Пользователь пишет, какой бизнес его интересует (пример: кофейня у дома, маникюр на дому, магазин одежды в ТЦ, продажа на маркетплейсе).
2. LogiQ собирает, анализирует и выдает структурированные рекомендации.

• Описание решения
• Что делает бот
LogiQ — AI-ассистент, который помогает владельцам микробизнеса ориентироваться, проводить диагностику и получать советы по маркетингу, финансам и процессам.

• Для кого
Для владельцев и будущих владельцев малого бизнеса (кофейни, маникюры, магазины одежды, маркетплейс-продавцы).

• Какие задачи решает

- С чего начать бизнес
- Анализ текущего состояния
- Консультации по ключевым бизнес-направлениям

• Как интегрирован LLM
LLM используется для анализа вводимой информации, генерации персонализированных рекомендаций и диалогового взаимодействия с пользователем.

• Архитектура

- Frontend: чат-бот интерфейс (Telegram)
- Backend: сервер с логикой бота на Python (main.py)
- Интеграция с LLM через API (Grok)
- Базы данных и вспомогательные сервисы для хранения и обработки данных
- Docker-контейнеризация для удобного развёртывания
